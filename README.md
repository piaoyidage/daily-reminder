# Daily Reminder

每日提醒，设置每天的提醒时间和提醒内容。

## Extension Settings

* `remind.onDuty`: set on duty time
* `remind.onDutyTip`: set on duty tip
* `remind.offDuty`: set off duty time
* `remind.offDutyTip`: set off duty tip
* `remind.defaultTip`: set default tip
